# Library Management System - Core Components (JS)

This project demonstrates the core architecture of a simple Library Management System using JavaScript and Node.js.

---

# Concepts Used

- ES Modules
- Factory Pattern
- Strategy Pattern
- Observer Pattern
- Dependency Injection
- Interface-like Abstractions
- Modular Design
